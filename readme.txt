Auto SRT Maker

Please download any model according to your requirement from https://alphacephei.com/vosk/models and unpack as 'model' in the current folder.
Install ffmpeg based on your operating system
install vosk using by following command
pip install vosk

copy the audio file or the video file to create srt into this folders and try the following commands in the termianl to make the srt


python .\test_srt.py .\output_audio.wav

or 

python .\test_srt.py .\output_video.mp4
